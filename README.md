# SwiftyAlgebra

Provides set of tools for managing numbers in Swift.

## How To Get Started

- Copy content of `Source` folder to your project.

or

- Use `SwiftyAlgebra` cocoapod

## Requirements

* iOS 9 and later
* Xcode 9 and later
* Swift 4
